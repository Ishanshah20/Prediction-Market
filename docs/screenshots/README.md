## How to capture

1. Run `cd frontend && npm run dev`
2. Open Chrome DevTools → Toggle device toolbar → iPhone 14 (390×844)
3. Screenshot each page
4. For CI/CD: push to `main`, go to Actions tab, screenshot the passing run
